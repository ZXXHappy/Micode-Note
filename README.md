# MiCode Notes (Modernized Fork) / MiCode 便签（现代化兼容分支）

## 中文简介
本项目基于 MiCode Notes 开源代码进行兼容性改造，目标是在新版 Android Studio 与 Android Gradle Plugin 环境下可正常编译、安装与运行。  
本仓库主要包含工程构建链路修复与兼容性调整，尽量保持原项目核心功能与使用体验。

## English
This project is a compatibility-focused fork of the open-source MiCode Notes.  
It is modernized to build, install, and run on recent Android Studio and Android Gradle Plugin versions, while keeping the original app behavior and purpose as much as possible.
